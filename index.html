Portfolio

Project Terbaru

Kami menghadirkan berbagai proyek pembuatan website yang responsif, modern, dan sesuai kebutuhan klien. Dengan teknologi terbaru seperti HTML, CSS, Javascript, React, Dan Laravel kami memastikan tampilan yang menarik serta pengalaman pengguna yang optimal.

Website Sekolah

Situs web sekolah yang dikembangkan menggunakan Laravel dengan panel admin berbasis FilamentPHP. Proyek ini mencakup halaman informasi publik bagi pengunjung (seperti profil sekolah, berita, galeri, dan kontak), serta halaman admin untuk mengelola data sekolah secara dinamis tanpa pengkodean.

Landing Page FOOD

Situs web makanan modern dan responsif yang menampilkan tata letak menu restoran. Dibuat menggunakan HTML & CSS dengan teknik Flexbox dan grid untuk mencapai desain yang bersih dan struktur yang ramah seluler.

Landing Page Kedai Coffe

Situs web kedai kopi yang bergaya dan responsif yang dibuat menggunakan HTML dan CSS. Menampilkan tata letak modern dengan Flexbox, efek hover yang halus, dan desain yang ramah seluler.

Landing Page Website Pertanian

Website landing page untuk sektor pertanian yang dirancang responsif dan informatif. Menampilkan informasi produk pertanian, keunggulan, serta kontak petani secara modern untuk menjangkau pasar digital.
